このリポジトリは、jhonyspicyが作成したWebアプリを公開・紹介するためのポータルページです。

## 公開中のアプリケーション

現在、以下のポーカー学習支援アプリを公開しています。

### 1. ポーカーパワーナンバードリル
- **URL**: [https://jhonyspicy.github.io/poker-power-number-drill/](https://jhonyspicy.github.io/poker-power-number-drill/)
- **概要**: ポーカーの主要なパワーナンバーを語呂合わせで覚えるためのアプリです。

### 2. ポーカーレンジ表の場所を覚える
- **URL**: [https://jhonyspicy.github.io/porker-range-chart-practice/](https://jhonyspicy.github.io/porker-range-chart-practice/)
- **概要**: レンジ表の中で自分のハンドがどのあたりに位置しているのかを瞬時にイメージできるようになるための練習アプリです。
